# Workforce Emotion & Attrition Intelligence

An AI-powered HR Analytics Platform that predicts employee attrition risk, performs workforce segmentation, and provides interactive business insights through an advanced Streamlit dashboard.

## Features

- Employee Attrition Prediction using Machine Learning
- Workforce Segmentation using K-Means Clustering
- Interactive HR Analytics Dashboard
- Attrition Risk Assessment
- Employee Cluster Analysis
- Workforce Trend Visualization
- Feature Importance Analysis
- Custom Dataset Upload Support

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Joblib

## Machine Learning Models

- Random Forest Classifier
- K-Means Clustering
- Feature Scaling using StandardScaler

## Dashboard Insights

- Attrition Probability Prediction
- Employee Risk Classification
- Salary vs Attrition Analysis
- Job Role Attrition Trends
- Overtime Impact Analysis
- Workforce Segmentation
- Correlation Heatmaps
- Feature Importance Visualization

## Dataset

IBM HR Analytics Employee Attrition Dataset
